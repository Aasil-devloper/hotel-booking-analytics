# Hotel Booking Analytics

## Project Overview
Analyzed 30,000 hotel booking transactions spanning April 2024–April 2025 to identify booking trends, cancellation drivers, revenue opportunities, and seasonality patterns.

## Key Findings
- Overall cancellation rate: 20.23%
- Failed booking rate: 7.53%
- Travel Agent bookings had the highest cancellation rate (27.9%)
- Standard Rooms had a 23.3% cancellation rate
- Web channel generated the highest average booking value (₹28,191)
- July–August were peak booking months
- Leisure travelers accounted for 60.4% of bookings

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Business Recommendations
- Reduce Travel Agent dependency
- Introduce non-refundable incentives
- Implement dynamic pricing during peak seasons
- Improve Mobile App user experience
- Launch loyalty programs for leisure travelers
